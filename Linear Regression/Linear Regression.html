Particionando Dados em Treino e Teste¶

Passo 4)

Agora vamos particionar o dataset em treino/teste. Para isso definimos "p=0.7", isto 70% da base ser escolhida aleatriamente para treino e 30% para teste do modelo.

Observe tambm que o "set.seed(x)" garante que ao replicarmos essa partio em outro computador por exemplo, os mesmos dados iro respectivamente prar treino e teste

In [8]:
set.seed(280)
trainIndex <- createDataPartition(df$SIZE, p = .7, list = FALSE)
In [9]:
dfTrain <- df[ trainIndex,]
dfTest  <- df[-trainIndex,]

Treinando Modelo Univariado¶

Passo 5)

Observe que em "method" definimos "lm" de linear model e que definimos varivel target sendo medv e inicialmente colocamos apenas a varivel PRATIO como preditora."

In [10]:

model_uni <- train(SIZE~AGE, data = dfTrain ,method = "lm")
